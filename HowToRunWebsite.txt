Download the following libaries
	a. Flask
	b. flask_mysqldb
	c. MySQLdb.cursors
	d. mysql.connector


The following steps are used to run the code.
1. activate the virtual enviroment named env using powershell (env\Scripts\activate)
2. compile code using powershell (python app.py) 
3. set app as flask ($env:FLASK_APP="app.py")
4. run flask (flask run)
5. Go to browser and enter the local address given on the powershell screen (127.0.0...)
6. Site should appear with the login page.
